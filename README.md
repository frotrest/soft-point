
<h1 align="left">🧩 Soft Point — Tailwind + Parcel</h1>

###

<p align="left">Современный одностраничный сайт, разработанный с использованием Parcel и Tailwind CSS. Подходит для презентации SaaS-продукта, стартапа или сервиса.</p>

###

<h2 align="left">🧑‍💻 Технологии сайта</h2>

###

• 	⚡️ Быстрая сборка и обновление через Parcel

• 	📦 Автоматическая оптимизация ресурсов (JS, CSS, изображения)

• 	🛠 Prettier + PostCss для чистого и аккуратного кода

•   📖 Использование фреймворка Tailwind + PostCss

• 	🚀 Готовый деплой на Netlify

 •  📁 Использование пакета мененджеров yarn

###

<h2 align="left">📁 Структура проекта</h2>

###

```
soft-point/
├── src/
│   ├── index.html        # Главная страница
│   ├── css/              # Tailwind стили
│   ├── js/               # Скрипты
│   └── img/           # Изображения и ресурсы
├── tailwind.config.js    # Конфигурация Tailwind
├── .postcssrc.json       # Настройки PostCSS
├── netlify.toml          # Конфигурация деплоя
├── package.json          # Скрипты и зависимости
└── README.md             # Описание проекта
```
